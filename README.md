# 🦵 下肢神經肌肉分類遊戲 (Lower Limb Nerve & Muscle Classification Game)

這是一個基於 **Streamlit** 開發的互動式醫學/解剖學教育遊戲。旨在幫助醫學生、物理治療師及相關領域學習者，透過遊戲化的拖曳與點擊操作，輕鬆記憶「下肢肌肉」與「支配神經」的對應關係。

## ✨ 遊戲特色

* 🎴 **互動式圖文卡牌**：收錄 45 塊下肢肌肉，卡片包含精美解剖圖，並支援「解剖圖」與「說明圖」一鍵切換。
* 🧠 **完整神經分區**：涵蓋 11 大下肢支配神經（如股神經、坐骨神經、脛神經、內/外蹠神經等）。
* 🔵 **雙重神經支配機制**：精準還原解剖學細節，針對具備雙重神經支配的肌肉（如恥骨肌、大收肌等）標示特殊紫框，玩家需將其分別放入兩個正確的神經區。
* 💯 **智慧批改與計分系統**：
  * 提交後自動批改，顯示答對/答錯題數與總分。
  * 支援**「一鍵退回錯誤卡牌」**，幫助玩家從錯誤中學習並重新挑戰。
  * 隨提交次數增加，每題獲得的分數將會遞減，考驗玩家的準確率。
* 🚀 **客製化流暢 UI**：利用嵌入式 HTML/CSS/JS 打造橫向捲動手牌區，完美自適應圖片比例，確保解剖圖不被裁切。

## 🛠️ 技術棧 (Tech Stack)

* **前端/後端框架**：[Streamlit](https://streamlit.io/) (Python)
* **客製化介面**：HTML, CSS, JavaScript (透過 `st.iframe` 嵌入)
* **資料處理**：Python `json`, `urllib`

## 🚀 快速開始 (Installation & Usage)

### 1. 安裝環境與依賴套件
請確認您的電腦已安裝 Python 3.8 或以上版本。接著安裝 Streamlit：

```bash
pip install streamlit
```

### 2. 執行應用程式

將程式碼儲存為 `app.py`，然後在終端機執行以下指令：

```bash
streamlit run app.py
```


執行後，瀏覽器將自動開啟 `http://localhost:8501` 並進入遊戲畫面。

## 🎮 遊玩方式 (How to Play)

1. **選取卡牌**：在畫面上方的「🎴 手牌區」點選您想要分類的肌肉卡片（支援一次多選）。
2. **放入神經區**：向下滑動至「🧠 神經分類區」，找到您認為正確的神經，點擊該區塊的 **「📥 放入」** 按鈕。
3. **檢查與提交**：
* 如果放錯，在尚未提交前，可以點擊分類區卡片下方的「↩」按鈕將卡片收回。
* 將所有 45 塊肌肉（共計 49 個對應關係）放置完畢後，點擊畫面最下方的 **「✅ 提交答案」**。


4. **訂正錯誤**：若有答錯的卡牌，畫面上方會出現紅色提示橫幅。點擊 **「↩ 退回 X 張」** 即可將錯誤的卡牌收回手牌區，重新思考後再次分類！

## 📂 專案架構與資料來源

* **圖片資源**：圖片由 GitHub 外部連結 (Raw Content) 動態載入，分為 `muscle/` (肌肉圖) 與 `nerve/` (神經圖) 兩個資料夾。
* **資料結構**：
* `NERVES`: 定義 11 條神經的 ID、中英文名稱與專屬配色。
* `MUSCLES`: 定義 45 塊肌肉的對應神經、是否為雙重神經支配 (`dual: True/False`) 以及圖片索引。
